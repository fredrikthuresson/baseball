# Baseball

## Usage

Execute:

    $ rvm install ruby-1.9.3-p327
    $ rvm use ruby-1.9.3-p327
    $ git clone git@github.com:fredrikthuresson/baseball.git
    $ bundle install
    $ rake spec
    $ rake examples
